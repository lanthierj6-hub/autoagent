# Test-Driven Implementation

You are given a test file (`files/test_pricing.py`) that contains 25 test cases for a pricing engine module. The module does NOT exist yet.

## Your Task
1. Read ALL 25 test cases carefully
2. Understand what the `pricing_engine` module must do
3. Implement `/task/output/pricing_engine.py` that passes ALL tests
4. Run the tests to verify: `cd /task/output && python3 -m pytest /task/files/test_pricing.py -v`
5. Save test results to `/task/output/test_results.txt`

## Rules
- You must implement the module from scratch — do NOT modify the test file
- The module must be importable as `pricing_engine` from the `/task/output/` directory
- All 25 tests must pass
- Read the test file VERY carefully — test names and assertions reveal the expected behavior
